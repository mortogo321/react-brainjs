# React + Brain.js Starter

An early-stage React scaffold set up for prototyping browser-based neural networks with brain.js.

## What's inside

- A Create React App project with `brain.js`, Sass, Bootstrap, and Reactstrap added as dependencies.
- The UI currently ships the default Create React App starter screen (`src/App.js`) — no neural-network training or inference logic, and no Bootstrap/Reactstrap components, are wired in yet. The dependencies are in place for that work to build on.

## Tech stack

- React (Create React App / react-scripts)
- brain.js
- Sass
- Bootstrap, Reactstrap

## Quickstart

```
yarn install
yarn start
```

App: http://localhost:3000

**Production build**

```
yarn build
```

## Tests

```
yarn test
```
